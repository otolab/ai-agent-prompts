# 参照パターンテストモード

様々な@参照パターンのテスト用です。

## パターン1: 行頭の@参照

@test_ref_file.md

## パターン2: インデントされた@参照

  @test_ref_file.md

## パターン3: リスト項目の@参照

- @test_ref_file.md

## パターン4: インデントされたリスト項目の@参照

  - @test_ref_file.md

すべてのパターンで参照ファイルが読み込まれるはずです。
