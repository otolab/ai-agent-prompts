## 使い方

### 基本的な使用方法

1. 目的に応じてカテゴリを選択
2. 適切なプロンプトファイルを参照
3. 必要に応じてカスタマイズ

### プロンプト選択ガイド

- **分析が必要な場合**: `core/analysis.md`
- **コンテンツ生成が必要な場合**: `core/generation.md`
- **既存コンテンツの改善が必要な場合**: `core/optimization.md`
